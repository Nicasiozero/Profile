Book Reviews

  • Boolean and double data type was only briefly mentioned in
    page 146 for data storage and never explored deeply
  • There’s not much explanation about special data type,
    other than the code example itself
  • Didn’t cover all data modifier, only partially
  • Only mention the format of Hexadecimal and Octadecimal, yet doesn’t fully
    explain what is Hexadecimal and Octadecimal
  • There’s no mention of variable name’s capital sensitivity
    and limitation (such as no start with number)
  • Not exploring into Bit field


Chapter 13-25 of this book covers GTK in depth,
while this note only briefly mentions it.
Overall, this book is basic, easy to read,
doesn’t go as in-depth, and full of beginner’s information.
Suitable for beginners who're starting to pick up coding,
yet lacking in advanced C code information.
